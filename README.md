# ue5-cpp-spellcasting

## Overview
This project demonstrates a Spell Casting gameplay system built in Unreal Engine (C++).

## Features
- Modular, player-facing gameplay logic
- Event-driven architecture
- Designer-friendly editor exposure


## Architecture
- Implemented using Actor Components
- Clean separation between gameplay logic and data
- Designed for reuse across multiple projects
- -Needs minimal setup ( link to player Class and Input wiring )
